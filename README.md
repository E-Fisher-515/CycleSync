# CycleSync

A modern web application for tracking and synchronizing menstrual cycles. Built with HTML, CSS, and JavaScript.

## Features

- Cycle tracking and logging
- Period prediction and reminders
- Symptom tracking
- Data visualization
- Privacy-focused local storage

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start tracking your cycles

## Development

This is a client-side only application that stores data locally in the browser for privacy.
